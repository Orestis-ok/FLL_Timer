# FLL Timer

A simple 2:30 minute timer made for FLL practice.

## What it does

- Starts a 2:30 countdown
- Plays a sound when starting
- Plays a warning sound at 0:30
- Plays a sound when time ends
- Changes color (green → yellow → red flash)

## How to run

Open `index.html` in a browser or use Live Server in VS Code.

## Files

- index.html
- style.css
- script.js
- start.mp3
- warning.mp3
- end.mp3
